How Twitch Integration Works

1

Creators Connect Twitch: Creators register their profile and securely connect their Twitch account via OAuth in their creator profile settings.

2

Admin Approves: Server admins review and approve creators for live notifications in this tab. This prevents spam and ensures quality content.

3

Automatic Monitoring: The bot checks approved streamers every 3 minutes for live status. No manual commands needed!

4

Rich Notifications: When a creator goes live, a customized Discord embed is posted with stream title, game/category, thumbnail, and viewer count.

Security: All OAuth tokens are encrypted using AES-256-GCM before storage. The bot only has read access to check live status.

Note: Twitch access tokens expire after ~60 days. The bot will automatically refresh them, but creators may need to re-auth if issues occur.
