How it works

Step 1

Add the bot, use the dashboard

Full XP, LFG, moderation, RSS, live alerts - everything works out of the box. No QuestLog account needed.

Step 2

Apply to the QuestLog Network

One application unifies your server with the QuestLog platform. Your guild gets a public profile and appears in the community directory.

Step 3

Members link their accounts

Fluxer XP merges into a unified QuestLog profile. One leaderboard, one level, across every platform they play on.

🖥️

Full Web Dashboard

Every bot feature has a matching web interface at casual-heroes.com/ql/dashboard/fluxer/. Configure XP settings, manage LFG, set up live alerts, moderate members, and view leaderboards - all from the browser with no slash commands required.

Self-Hosting

QuestLog Bot for Fluxer is fully open source under the AGPL-3.0 license. Clone the repo, set your Fluxer bot token, point it at a MySQL database, and you are running your own instance. Full setup instructions are in the README.
