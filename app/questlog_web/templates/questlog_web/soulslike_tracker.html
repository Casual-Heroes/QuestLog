What's included

Death Tracking

Log deaths manually with F9 or tap on the web tracker. Session deaths, survival time, Deaths/Boss all tracked.

Boss Tracker

All 247 ERR bosses / 236 ER bosses. Tap to set as Fighting, tap again to mark Defeated. Rage decays by boss tier.

Rage / Hollow System

Rage builds with deaths, decays with boss kills. Five states: Maiden's Grace, Staggered, Frenzied, Cursed, HOLLOW.

QuestLog Sync

Login with your QuestLog account. Runs, boss kills, and stats sync live to the web. Pick up on phone or web anytime.

Stream Overlays

Browser sources for OBS/Meld. Deaths, rage bar, boss tracker, fighting indicator. Updates in real time via QuestLog.

Web Tracker (Console / Mobile)

Same run on your phone, tablet, or second screen. Console players can use the web tracker without the app.

Stream Overlays

Once connected to QuestLog, your run page shows overlay URLs. Add them as Browser Sources in OBS or Meld - they update live as you play.
