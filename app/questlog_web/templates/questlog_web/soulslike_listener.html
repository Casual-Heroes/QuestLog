FAQ

Is this safe? Will it get me banned?
The Listener only reads your screen (OCR) and listens for key presses. It does not inject into the game or modify any game files. It is functionally equivalent to you manually pressing a button.
Where is my session token?
Start a Mortality Run from the QuestLog Souls Tracker page. Your session token is in the Web Tracker URL: questlog.casual-heroes.com/soulslike/runs/YOUR_TOKEN/
OCR says it needs to download something?
EasyOCR downloads its language model (~100MB) on first run. This is a one-time download. If you skip it, you can still use F9/F10 hotkeys manually.
Can I use this on multiple monitors?
The OCR scans your primary monitor by default. Game window detection ensures it focuses on the right screen when the game is in the foreground.
