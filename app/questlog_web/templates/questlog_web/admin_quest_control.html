Warns players before the backup time above, then stops the server, takes a full backup while offline, restarts it, and confirms online. Replaces the live in-place backup entirely for this instance while enabled. All timings below are fully under your control.

Warning lead times

Each chip below is a warning that goes out to your notice channels that many minutes before the "Backup time" set above. E.g. with 15/10/5 min chips and a 23:30 backup time, players get warned at 23:15, 23:20, and 23:25 - then the server actually stops at 23:30. Click the on a chip to remove it, or add your own below.

Cycle timing

The automated cycle is: stop server → wait → take backup → wait → start server → wait → confirm online message. Each pause below gives that step time to actually finish before the bot moves on - too short and it may take the backup mid-shutdown or announce "online" before the server has finished booting.
